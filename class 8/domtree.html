📋 Combinator Reference

Combinator Symbol Meaning Example
Descendant ␣ (space) Any level inside .hero p
Child > Direct children only .hero > p
Adjacent Sibling + Immediately next sibling h2 + p
General Sibling ~ All following siblings h2 ~ p

⚠️ Common Mistakes: Combinators

  • Confusing descendant (space) with child (>). The space reaches all the way down. The > stops at one level.
  • Forgetting that sibling selectors (+ and ~) only target elements that come after the first element in the HTML source order — not before.
  • Using overly long combinator chains like .page .main .content .card .title. This creates fragile selectors that break when the HTML structure changes. Keep combinator chains short and intentional.
